# Escalation Rules

If a customer issue cannot be resolved using standard procedures or known resolutions:
1. Tier 2 Escalation: Escalate to Tier 2 support for complex billing queries, security vulnerability reports, or persistent API errors.
2. Tier 3 Escalation: Escalate to Tier 3 engineering for severe service outages, data corruption, or unresolved system bugs.
3. Priority Customers: Enterprise tier customers should be fast-tracked to the next tier immediately without waiting.
